# Sticky Note

A small corner curl works well for reminders or in-product tips.

import { PeelWrapper, PeelTop, PeelBack, PeelBottom } from "react-peel";

## Implementation

```tsx
import { PeelWrapper, PeelTop, PeelBack, PeelBottom } from "react-peel";

interface StickyNoteProps {
  message: string;
  hiddenMessage?: string;
  width?: number;
  height?: number;
}

export function StickyNote({
  message,
  hiddenMessage = "Hidden note",
  width = 200,
  height = 200,
}: StickyNoteProps) {
  return (
    <PeelWrapper preset="stickyNote" height={height} width={width} drag>
      <PeelTop style={{ backgroundColor: "#fff9c4", padding: 16 }}>
        {message}
      </PeelTop>
      <PeelBack style={{ backgroundColor: "#fff59d" }} />
      <PeelBottom
        style={{
          backgroundColor: "#f5f5f5",
          padding: 16,
          display: "flex",
          alignItems: "center",
          justifyContent: "center",
          fontStyle: "italic",
        }}
      >
        {hiddenMessage}
      </PeelBottom>
    </PeelWrapper>
  );
}
```

## Usage

```tsx
<StickyNote message="Call Alex" />
```

## Styling Tips

- Use pastel colors for the top layer.
- Add padding and a subtle box shadow for depth.
- Consider a handwritten font for the top layer text.
